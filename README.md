# Hunter Camp Portfolio

A Monster Hunter-inspired interactive portfolio built as an experiment to see how far modern AI-assisted development can go.

This project started as a test: could an AI help build a full interactive web experience with a cinematic intro, camp UI, 3D models, music selection, themed menus, and a game-like atmosphere?
The result was honestly impressive. Most of the structure, animations, UI logic, and iteration process were built with AI assistance, then reviewed, adjusted, and corrected manually.

The goal was not to create a commercial product, but to explore how AI can support creative front-end development while keeping a strong visual direction.

## Concept

The portfolio is presented as a hunter camp.

Instead of a classic developer portfolio layout, the visitor enters a camp, interacts with a Felyne guide, explores a Monster Hunter-style menu, opens a Guild Card-inspired profile panel, and can change the camp music.

The project is designed as a fan-made tribute to the atmosphere, UI language, and world feeling of the Monster Hunter series.

## Features

* Cinematic camp entry animation
* Monster Hunter-inspired UI
* Theme switcher between an Iceborne-inspired style and a Wilds-inspired style
* Interactive Felyne guide
* 3D scene using Three.js
* Campfire 3D model
* Hunter profile popup
* Camp music item box
* Random camp music playback
* Music persistence through local storage
* Language toggle between French and English
* Thank-you / credits page with rolling credits

## Technologies Used

* HTML
* CSS
* Vanilla JavaScript
* Three.js
* GLTFLoader
* Font Awesome

## AI-Assisted Development

This project was built as an AI-assisted development experiment.

The purpose was to test how much modern AI could help with:

* structuring a creative web project
* writing and refactoring JavaScript
* building complex CSS layouts
* handling animations and UI states
* integrating 3D models
* iterating quickly on visual direction
* debugging interaction issues

The AI helped generate a large part of the code, but the final result required manual testing, corrections, asset preparation, design decisions, and implementation adjustments.
